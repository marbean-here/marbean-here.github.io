Internship and Placement Tracking System

A web-based system for students to find internships or placements and track their applications, while allowing admins to manage data efficiently.

PHP HTML CSS JavaScript MySQL/XAMPP Web Application

Project Overview

This system allows students to browse available internship and placement opportunities, submit applications, and track their status. Admins can manage student accounts, company data, post new opportunities, and approve or reject applications. The platform ensures organized and efficient management of applications while providing students with an easy-to-use interface for career opportunities.

Key Features

  • Students can view and search for internship and placement opportunities.
  • Students can track the status of their applications (pending, accepted, rejected).
  • Admins can manage student accounts and company data.
  • Admins can post new internship and placement opportunities.
  • Admins can accept or reject student applications.
  • User-friendly dashboards for both students and admins.

My Role

I designed and implemented the full system independently, handling the database structure, backend logic, front-end interface, and user role management for both students and admins. Every aspect of the project, from functionality to interface design.

Challenges & Solutions

  • Managing database relationships – solved by creating clear MySQL tables for students, companies, internships, placements, and applications.
  • Maintaining application data consistency – implemented validation and proper CRUD operations.
  • Ensuring interface clarity – designed separate dashboards for students and admins to avoid confusion.

What I Learned

  • Full-stack web development using PHP, MySQL, JavaScript, HTML, and CSS.
  • Database design and management with XAMPP/MySQL.
  • Implementing user roles and permissions in a web application.
  • Handling dynamic data and application logic for tracking submissions.
  • Designing user-friendly interfaces for multiple types of users.

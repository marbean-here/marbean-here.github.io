Pen Vending Machine (Arduino + Flutter)

A prototype mobile-controlled pen vending machine where the admin can remotely add credit and dispense pens using a Flutter app connected via Bluetooth to Arduino hardware.

Arduino Flutter Bluetooth IoT Hardware Prototype

Project Overview

A mobile-controlled pen vending machine prototype using Arduino and Flutter. The system integrates hardware components (LCD, servo motor, buzzer, and button) with a mobile app for the admin. The admin can add credits and remotely dispense pens via Bluetooth.

Key Features

  • Hardware prototype includes LCD, buzzer, push button, and servo motor for pen dispensing.
  • Mobile app (Flutter) allows the admin to add credit and dispense pens remotely.
  • Bluetooth connectivity using HC-05 module for real-time communication.
  • Simple and intuitive admin interface for managing the vending machine.
  • System ensures safe and controlled dispensing of pens.

Technology Stack

My Role

Designed and built the full system, including hardware prototype design, Arduino programming, and Flutter mobile app development. Every aspect, from coding to hardware integration.

Challenges & Solutions

  • Integrating Arduino hardware with mobile app — solved by using HC-05 Bluetooth module for communication.
  • Ensuring accurate servo operation — calibrated servo motor to dispense a single pen reliably.
  • User interface design — created an intuitive Flutter app that clearly shows credit balance and dispense button.
  • Managing real-time updates — handled synchronization between app and hardware using Bluetooth commands.

What I Learned

  • Microcontroller programming with Arduino (C/C++).
  • Flutter app development and connecting it to hardware via Bluetooth.
  • Integrating multiple hardware components (LCD, servo, buzzer, button) into a single functional prototype.
  • Designing a functional IoT system for real-time remote control.
  • Troubleshooting communication and timing issues between hardware and mobile app.
